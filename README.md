# Line-follower-Robot
Robot which is made up of Arduino controller ,infrared sensor , motor controller and DC motor  . Which simply follows the black line at a faster rate


Two codes are mentioned above 

1 . Speed Adjustment code : By this code we can able to adjust the working speed of the robot .By activating the enable pin in the  REES52 DC Driver Controller

2. Const speed code : Speed of the robot is constant in this case but the accuracy is little higher then the previous one
